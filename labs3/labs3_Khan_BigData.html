Мета роботи

Освоїти сучасні методи обробки неструктурованих та напівструктурованих даних (JSON), навчитися перетворювати формати таблиць (широкий ↔︎ довгий) за допомогою tidyr, а також застосовувати віконні функції.

Вступ

У реальному світі інженерії даних вихідні дані часто надходять у нестандартних форматах. Ця лабораторна робота охоплює три критичні аспекти: Робота з JSON, Перетворення форматів та Віконні операції.

prepared_daily <- daily_metrics %>%
  mutate(month = substr(date, 1, 7), conversion = unique_users / page_views)

windowed_data <- prepared_daily %>%
  arrange(date) %>%
  group_by(month) %>%
  mutate(
    roi = (unique_users * 0.1 - cost) / cost,
    roi_rank = dense_rank(desc(roi)),
    bounce_change = bounce_rate - lag(bounce_rate, default = first(bounce_rate)),
    cum_users = cumsum(unique_users)
  ) %>%
  ungroup()

combined_data <- windowed_data %>%
  left_join(monthly_traffic, by = "month", relationship = "many-to-many")
